# 관리자 페이지 구성 및 기능 정리

## 1. 기본 관리자 페이지
/admin/login
- 관리자 로그인 기능
- 로그인 후 기업 목록 페이지로 이동

/admin/profile
- 관리자 정보 수정
- 비밀번호 변경


## 2. 기업 관리
/admin/companies/list
- 전체 기업 목록 조회
- 기업 검색 기능
- 기업 필터링 (상태, 업종 등)
- 페이지네이션

/admin/companies/create
- 새 기업 정보 등록
- 업종 선택 (모달)
- 필수 정보 유효성 검사

/admin/companies/edit/:id
- 기존 기업 정보 수정
- 업종 수정 (모달)
- 수정 이력 관리

/admin/companies/:id
- 기업 상세 정보 조회
- 구성원 목록 조회
- 기업 삭제 (모달 확인)


## 3. 승인 관리
/admin/approvals/companies/list
- 기업 승인 요청 목록
- 승인 상태별 필터링
- 기업 → 신청자 순 정보 출력
- 페이지네이션

/admin/approvals/companies/:id
- 기업 승인 요청 상세 정보
- 승인/반려 처리 (모달)
- 반려 시 사유 입력

/admin/approvals/members/list
- 구성원 승인 요청 목록
- 승인 상태별 필터링
- 신청자 → 기업 순 정보 출력
- 페이지네이션

/admin/approvals/members/:id
- 구성원 승인 요청 상세 정보
- 승인/반려 처리 (모달)
- 반려 시 사유 입력


## 4. 회원 관리
/admin/members/list
- 전체 회원 목록
- 회원 검색 기능
- 상태별 필터링
- 회원 내보내기 (엑셀)
- 페이지네이션

/admin/members/create
- 새 회원 정보 등록
- 필수 정보 유효성 검사
- 중복 확인 (이메일 등)

/admin/members/edit/:id
- 기존 회원 정보 수정
- 상태 변경
- 수정 이력 관리


## 공통 기능
1. 모든 목록 페이지
   - 검색
   - 필터링
   - 정렬
   - 페이지네이션

2. 모달 처리
   - 업종 찾기
   - 승인/반려 확인
   - 삭제 확인

3. 데이터 처리
   - 입력값 유효성 검사
   - 에러 처리 및 표시
   - 작업 완료 알림

4. 보안
   - 관리자 권한 확인
   - 페이지별 접근 제어
   - 로그인 세션 관리
